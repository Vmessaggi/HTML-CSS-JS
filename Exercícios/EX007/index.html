Principais formatações:

Negrito / Destaque-

Nesta frase temos um termo em negrito usando a tag <b>

Nesta frase temos um termo em destaque usando a tag <strong>

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i>

Nesta frase temos um termo em ênfase usando a tag <em>

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>.

Outro texto marcado mas não herda a cor e volta pro padrão.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag <u>

Texto sobrescrito

Para inserir coisas do tipo X20+3 com a tag <sup>

Texto subscrito
